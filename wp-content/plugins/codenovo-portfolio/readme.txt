== Description ==

Codenovo portfolio is used for showing your projects/works/portfolio in an awesome way. They can be arranged in a nice gallery format with jquery filters. Every portfolio can be categorized. You can easily put title, description and other information. All portfolios are filterable. You can find some similar plugin with filter option, but this plugin is handier than others. Comfortable to use, easy to navigate and very simple to build.

Please check the [Demo](http://cnfolio.codenovo.com/demo/ "Codenovo Portfolio") for real understanding.

The unique part of this plugin is a widget. Codenovo plugin offers a portfolio widget as well. You can showcase your portfolio in anywhere through this widget. You can showcase your top portfolio or any portfolio you choose or even ALL.


== Features ==

1. Easy to customize
2. Smooth filtration with jQuery.
3. All information can be put like Portfolio title, description, project name, project live URL.
4. All portfolios can be categorized, even to multiple categories.
5. Portfolio Widget to showcase your portfolio anywhere you want.
6. Shortcode for pages/posts editor- [codenovo-portfolio]
7. PHP code for template file- codenovo_portfolio();

== Installation ==

Using the WordPress Dashboard

1. Navigate to the 'Add New' in the plugin dashboard
2. Search for 'Codenovo Portfolio'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

Uploading in WordPress Dashboard

1. Navigate to the 'Add New' in the plugin dashboard
2. Navigate to the 'Upload' area
3. Select codenovo-portfolio.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

Using FTP

1. Download codenovo-portfolio.zip
2. Extract the codenovo-portfolio directory to your computer
3. Upload the waving-portfolio directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= Is it free? =

Yes, it is completely free.

= How can I add a portfolio? =

In the portfolio section click "ADD NEW" and fill out the necessary information and click "PUBLISH".

= How can I add portfolio Image? =

In the "ADD NEW" section, there is option to add "Set Featured Image". Set an image there. This image will be your portfolio image.
